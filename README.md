# ScholarshipCoach

## How to run

1. Create and activate a Python 3.11+ virtual environment.
2. Install dependencies:
   - `pip install -e .[dev]`
3. Run tests:
   - `pytest -q`
4. Start the app (placeholder):
   - `streamlit run app/main.py`

## Project structure

- `app/`
- `src/`
- `scripts/`
- `tests/`
- `docs/`
- `data/raw/`
- `data/processed/`
